# BiliTask

B站每日任务自动化脚本 — 扫码登录，一键完成每日登录/观看/投币/分享。

## 功能

- 扫码登录（桌面生成二维码，手机B站App扫码）
- 每日登录 +5 EXP
- 观看视频 +5 EXP
- 投币 +10 EXP/枚（1-5枚可选，自动分配视频）
- 分享视频 +5 EXP（移动端模拟）
- 定时自动运行（每天早7点）
- 自定义主题色 + 暗色模式
- 经验升级预估

## 快速开始

```bash
npm install
npm start
```

浏览器打开 `http://localhost:8000`，扫码登录，勾选任务，点击"立即运行"。

## 投币策略

| 选择 | 分配 |
|------|------|
| 1 枚 | 1个视频 ×1 |
| 2 枚 | 1个视频 ×2 |
| 3 枚 | 2个视频 2+1 |
| 4 枚 | 2个视频 2+2 |
| 5 枚 | 3个视频 2+2+1 |

## 注意事项

- `config.json` 包含登录 cookie，已被 .gitignore 排除
- 首次运行需先扫码登录
- 分享功能仅模拟APP端请求，B站可能不计入经验
- 投币从每日热门视频池中自动选择
